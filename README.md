# 💳 Credit Card Approval Prediction System

**IBM SkillsBuild UE AICTE Internship – June 2026**

## 📌 Overview

This project uses Machine Learning to predict whether a credit card application is likely to be approved or rejected based on an applicant's demographic and financial information. It demonstrates data preprocessing, exploratory data analysis (EDA), model training, evaluation, and a simple Flask web application for prediction.

---

## 🚀 Features

- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Multiple Machine Learning Models
- Model Performance Comparison
- Flask-based Web Interface
