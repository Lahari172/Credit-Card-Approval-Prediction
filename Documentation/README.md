## Credit Card Approval Prediction System

### IBM SkillsBuild UE AICTE Internship – June 2026

---

## Project Description

The Credit Card Approval Prediction System is a Machine Learning project developed to automate the process of evaluating credit card applications. The system predicts whether an applicant is likely to be approved or rejected based on demographic and financial information.

The project demonstrates the complete Machine Learning workflow, including data preprocessing, exploratory data analysis (EDA), feature engineering, model training, model evaluation, and deployment using a Flask web application.

---

## Problem Statement

Banks receive thousands of credit card applications every day. Manually reviewing each application is time-consuming and prone to human error. This project aims to automate the approval process by using Machine Learning algorithms to classify applicants as eligible or ineligible for credit card approval.

---

## Objectives

- Automate the credit card approval process.
- Perform data preprocessing and feature engineering.
- Analyze applicant data using visualization techniques.
- Train multiple Machine Learning classification models.
- Compare model performance.
- Deploy the best-performing model through a Flask web application.

---

## Dataset

The project uses the **Credit Card Approval Prediction** dataset containing:

- application_record.csv
- credit_record.csv

The dataset includes applicant demographic information, financial details, employment records, and historical credit payment status.
